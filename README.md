# Galaxy Defender

Galaxy Defender is a 2D space shooter being developed in Unity using C#. The project began as an exploration of Unity's game development workflow and is gradually evolving into a complete arcade-style shooter.

## Current Progress

The current build includes:

- Player movement
- Shooting mechanics
- Bullet prefabs
- Keyboard controls

Additional gameplay systems such as enemies, collision handling, scoring, and progression are currently under development.


## Tech Stack

- Unity
- C#
- Visual Studio
- Git

## Development Notes

The project is being developed incrementally, with each feature implemented and tested separately before integration into the main game loop.

Recent work has focused on:
- Input handling
- Player controls
- Projectile mechanics
- Prefab-based object creation

Upcoming work includes:
- Enemy spawning
- Collision detection
- Score tracking
- Gameplay balancing
